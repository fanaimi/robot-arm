# robot-arm_keyboard-control

Quick tests to control a robot arm using keyboard. More coming soon for mobile control with GUI.


#### Using Arrow Keys, R, F, T, G

| KEY | ACTION |
| --- | --- |
| UP | extend forward |
| DOWN | extend backward |
| | |
| LEFT | rotate to left|
| RIGHT | rotate to right |
| | |
| R | lower forearm  |
| F | raise forearm |
| | |
| T | open claw |
| G | close claw |
